# Neovim Config

## Supports

- JS/TS/React
- HTML/CSS/TailwindCSS
- Lua
- JSON
- Easily Extendible

## IDE Plugins Tools

- Null-ls: Formatter
- Prettier
- LSP: with completion and snippets
- Telescope

## Other Plugins

- Sneak s + Better f/t
- Toggleterm
- Todo-comments highlighting
- React: comments, html tags suport
- Heavily uses mini.nvim to avoid bloat
